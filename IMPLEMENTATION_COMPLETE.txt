📊 CLASSE VALIDATEUR
═══════════════════════════════════════════════════════════════════════════

  Utilisation simple:

  // Valider un email
  $email = Validateur::email($_POST['email']);
  if (!$email) {
      $erreurs = Validateur::getErreurs();
  }

  // Nettoyer pour affichage
  echo echapper($_GET['nom']);

  // Générer un slug
  $slug = Validateur::slug($titre);

  ├── chaine()        → Chaine validée ou FALSE
  ├── entier()        → Int validé ou FALSE
  ├── decimal()       → Float validé ou FALSE
  ├── url()           → URL validée ou FALSE
  ├── date()          → Date validée ou FALSE
  ├── fichier()       → Info fichier ou FALSE
  ├── echapper()      → HTML escaped
  ├── nettoyer()      → Texte propre
  └── slug()          → URL-friendly

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

  Problem: Colonnes manquantes en BD
  Solution: Exécutez migrations/001_add_email_verification.sql
            mysql -u root -p ade_market < migrations/001_add_email_verification.sql

  Problem: Email non envoyé
  Solution: Vérifiez SMTP_* dans .env
            Exécutez test_inscription.php pour diagnostiquer

  Problem: "Token invalide"
  Solution: Token expire après 24h
            Allez sur /client/renvoyer_email_confirmation.php

  Problem: Erreurs de validation
  Solution: Vérifiez les critères de chaque champ
            Regardez les logs dans /logs/

  Problem: Permissions manquantes
  Solution: chmod 755 logs/
            chmod 755 migrations/
